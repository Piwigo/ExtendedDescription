[photo id=xx]

이 태그를 사용하면 원하는 크기의 사진을 한 장 삽입할 수 있습니다.

옵션:
  • id 사진 식별자
  • album (선택 사항) 상위 앨범 식별자
  • size (선택 사항) 사진 크기 (SQ, TH, XXS, XS, S, M, L, XL, XXL 중 선택)
  • html (선택 사항) false로 설정 시 HTML 없이 사진 URL만 반환
  • link (선택 사항) true로 설정 시 사진이 사진 페이지 링크로 감싸짐
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

위와 동일한 옵션이 제공되나, 사진은 전체 갤러리 또는 album에서 무작위로 선택됩니다.

여러 장의 사진을 표시하려면 다음 옵션을 사용하십시오:

  • nb_images (선택 사항) 표시할 이미지 수
[random]

[random album=123 size=M html=yes link=yes]

[random album=123 size=M html=yes link=yes nb_images=8]

[cat=xx]

이 태그는 설명에 앨범을 삽입하며, 여기서 xx는 앨범 식별자입니다.
