[slider album=xx]

이 태그는 사진 슬라이드쇼를 삽입할 수 있게 합니다.

옵션: (album 또는 list 중 하나를 설정해야 합니다)
  • album: (선택 사항) 원본 앨범
  • nb_images: (선택 사항) 슬라이드쇼에 표시할 최대 사진 수
  • random: (선택 사항) 앨범 내 사진 무작위 선택
  • list: (선택 사항) 쉼표로 구분된 사진 ID 목록
  • size: (선택 사항) 사진 크기, (SQ, TH, XXS, XS, S, M, L, XL, XXL) 중 선택
  • speed: (선택 사항) 슬라이드쇼 속도 (초 단위)
  • title: (선택 사항) 사진 제목 표시
  • effect: (선택 사항) 전환 효과 (자세한 내용은 NivoSlider 문서 참조)
  • arrows: (선택 사항) 탐색 화살표 표시
  • elastic: (선택 사항) 슬라이드쇼 크기를 각 사진에 맞게 조정
  • control: (선택 사항) 탐색 컨트롤 표시, thumb로 설정 가능
  • thumbs_size: (선택 사항) control=thumb 시 썸네일 크기(픽셀)
