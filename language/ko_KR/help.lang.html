다국어 설명은 [lang=xx] 및 [/lang] 태그 사이에 배치되며, 여기서 xx는 언어 코드입니다(예: en, fr, es, ...).

[lang=en]Default description[/lang]
[lang=fr]Description en français[/lang]
[lang=de]Deutsche Beschreibung[/lang]
[lang=ko]기본 설명[/lang]

기본 언어 설정

연결된 사용자가 선택한 언어로 설명이 제공되지 않을 경우 기본 설명이 사용됩니다.
[lang=default] 태그가 존재하지 않을 경우, [lang] 태그 외부 모든 내용이 기본 설명으로 간주됩니다.

[lang=default]Default description[/lang]
[lang=fr]Description en français[/lang]

// 또는

Default description
[lang=fr]Description en français[/lang]

모든 언어

[lang=all] 안에 있는 모든 내용은 연결된 사용자 정의 언어와 상관없이 설명에 포함됩니다.
이는 주로 설명 안에 HTML/JavaScript 코드를 포함하는 데 유용합니다.

[lang=all]<p>[/lang]
  [lang=default]Default description[/lang]
  [lang=fr]Description en français[/lang]
  [lang=de]Deutsche Beschreibung[/lang]
[lang=all]</p>[/lang]
