[photo id=xx]

תגית זו מאפשרת לך להוסיף תמונה אחת בגודל שתרצה.

אפשרויות:
  • id מזהה התמונה
  • album (אופציונלי) מזהה האלבום הראשי
  • size (אופציונלי) גודל התמונה, מתוך (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (אופציונלי) אם מוגדר כ-false התג מחזיר רק את כתובת ה-URL של התמונה, בלי HTML
  • link (אופציונלי) אם מוגדר כ-true התמונה מוקפת בקישור לדף התמונה
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

ישנן אותן אפשרויות כמו למעלה, פרט שהתמונה או התמונות נבחרות באקראי מתוך הגלריה כולה או מתוך album.

להצגת מספר תמונות השתמש באופציה:

  • nb_images (אופציונלי) מספר התמונות להציג
[random]

[random album=123 size=M html=yes link=yes]

[random album=123 size=M html=yes link=yes nb_images=8]
